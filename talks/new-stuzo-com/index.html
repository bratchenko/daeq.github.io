New Stuzo.com

Reimplement our company site.

And use it as a sandbox for experiments.

Goals of the project

  • Make site responsive.
  • Make site faster.
  • Make it a sandbox for developers.

Responsive site

  • There should be only 1 version of the site.
  • It should look and work great on all screens and devices.
  • It should adapt to device capabilities and restrictions.

Faster site

  • User should see content within 1 second from opening link.
  • We should monitor real-world site performance and improve it.

Sandbox for developers

  • Try new tools and technologies.
  • Experiment.
  • Refactor code to perfection.
