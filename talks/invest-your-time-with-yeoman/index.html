When do we need automation?

Walk or use transportation?

I'm new to the city.
My home is 20Km from the office.
Should I walk or should I use some kind of transport?

Walk

  • I know how to do it.
  • I always used walk before.
  • It's only 4-5 hours.

Use transport

  • Need to learn how to use transport.
  • Need to find out which route to use.
  • Need money.
  • Some people say it's faster.

Automation is useful when it saves more time than it takes.

Idea of Yeoman

  • Generators for automation of writing code.
  • Grunt for automation of routine tasks.
  • Bower for automation of dependency management.

Generators

  • Start project with appropriate generator
  • Generate files and components
  • Write your own generators

Good for both frontend and backend.

Grunt

  • Develop with development server.
  • Watch for changes.
  • Build for production.
